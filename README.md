# Java Mini Projects ☕🚀

This repository contains a collection of **Java mini projects** created to practice and strengthen core Java concepts such as OOP, collections, file handling, exception handling, and problem-solving.
These projects are small but practical implementations designed to improve Java programming skills and understanding of real-world programming patterns.

---

## 📌 Purpose

The goal of this repository is to:

* Practice **Core Java concepts**
* Improve **problem-solving and coding skills**
* Build a **strong foundation for larger Java projects**
* Maintain a collection of **hands-on practice projects**

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/hasinimaddula26-tech/My-Mini-java-projects/
```

2. Navigate to the project folder

```bash
cd java-mini-projects
```

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```
